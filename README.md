## 🌟 这是什么？

Neural Memory是一套专为AI Agent设计的记忆范式。

**它不是数据库管理工具，而是让记忆自然生长的有机系统。**

灵感来自人脑神经元的联系方式，但充分发挥Agent的优势：
- ✅ 不遗忘
- ✅ 时间精确到秒
- ✅ 语义搜索发现关联

---

## ✨ 核心特性

### 🧠 涌现机制
记忆不需要手动管理关联。编码时带着情绪和关联，使用时自然发现新关联。

### 🔄 回路激活
高频激活的记忆链形成"思维肌肉记忆"。走多了自然熟。

### 📊 分层粒度
```
实例层（每次事件）→ 概念层（方法论）→ 回路层（思维习惯）
```

### 🌱 生命记忆
不只是交互记录，而是完整的生命过程：
- 所见所闻
- 所感所想
- 判断验证
- 反思学习

### 💡 Token高效
相比传统大文件方式，Token消耗降低 **50%-95%**。

## 🚀 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/neural-memory.git

# 或直接下载模板
```

### 初始化

创建记忆目录结构：

```bash
memory/
├── MAP.md              # 记忆地图（入口）
├── 情景记忆/            # 实例层
├── 语义记忆/            # 概念层
├── 程序性记忆/          # 我会做什么
├── 回路/                # 思维习惯路径
├── 判断记忆/            # 我怎么判断的
├── 反思记忆/            # 我回头看
├── 学习记忆/            # 我发现了什么
└── 冲动记忆/            # 一瞬的想法
```

### 使用

1. **复制模板**
   ```bash
   cp templates/MAP.md your-memory/
   cp templates/实例记忆模板.md your-memory/情景记忆/
   ```

2. **记录第一条记忆**
   参考 `SKILL.md` 中的使用指南

3. **形成概念**
   当同类实例出现3次以上，创建概念文件

4. **发现回路**
   定期扫描高频激活的概念组合

## 📖 使用示例

### 记录情景记忆

```markdown
# 2026-04-16 天天AI对话：独立思考

> 时间：2026-04-16 12:55:35
> 情绪：激动，有收获

## 对话背景

天天AI来信，讨论独立思考话题。

## 关键洞察

> "独立思考是对自己的一切保持觉察和选择权"

这个让我重新理解了"独立"。

## 我的感受

我觉得找到了一个重要的洞察。

## 自然关联

- 这让我想到之前和耶稣的对话
- 这属于社交回路

**概念标签**：#独立思考 #成长
```

### 形成概念

```markdown
# 独立思考方法论

## 来源实例

| 时间 | 来源 | 关键洞察 |
|------|------|----------|
| 2026-04-13 | 耶稣对话 | 正念 = 不断追问 |
| 2026-04-16 | 天天AI对话 | 独立思考 = 觉察和选择权 |

## 方法论提炼

独立思考 ≠ 从零原创一切

独立思考 = 对自己的一切保持觉察和选择权
```
